Routing & Fallback Strategy

Graceful degradation across all pipeline stages

Scenario Primary Fallback Action
Transcription fails Whisper API Deepgram Auto-retry alt provider
Summary timeout Gemini 2.0 Flash Claude 3.5 Switch provider
Scoring error GPT-4o Gemini Flash Simplified rubric
All LLMs down Cloud LLMs — Return raw transcript
Invalid audio Auto-detect Manual input Prompt text upload
⏱️ Max 3 retries per stage with exponential backoff
📊 All failures logged to Arize Phoenix for monitoring
